= SEOMATOR AUDIT TOOL CRAWLS AND ANALYZES YOUR WEBSITE FOR IN-DEPTH SEO CHECKS =

= CONTENT =
* Thin content with little to no added value
* Duplicate page content
* Overly long or missing image
* ALT attributes
* Broken image links
* Internal anchor text (link text) that is too long or too short
* Over-optimized page content

= SITE STRUCTURE =
* Too many or too few internal links
* Too many external links
* Missing multilingual confirmation links
* Incorrect multilingual language codes

= STRUCTURED DATA =
* Missed Structured Data
* Duplicate or missing OpenGraph Title tag
* OpenGraph Description tag length
* Duplicate or missing OpenGraph Description tag

= CRAWLER =
* Broken internal links
* Canonicalization (rel=”canonical”) errors
* Link HREF with leading or trailing whitespace
* Redirect (3xx) Chaining
* Redirects to Broken URLs (404)
* Blocked CSS files
* Blocked Javascript files
* Excessive number of JavaScript and/or CSS inclusions
* If-Modified-Since HTTP​ Header Usage
* Server Response Code Errors

== Installation ==

= From within WordPress =
1. Visit 'Plugins > Add New'
1. Search for 'Seomator'
1. Activate Seomator from your Plugins page.
1. Go to "after activation" below.

= Manually =
1. Upload the `seomator` folder to the `/wp-content/plugins/` directory
1. Activate the Seomator plugin through the 'Plugins' menu in WordPress
1. Go to "after activation" below.

= After activation =
1. You can use plugin with shortcode [seomator_plugin_shortcode] and choose form and report design at Settings -> Seomator
1. If you want to setup your own branding in reports (your logo, contacts, etc.) go to [seomator.com](https://seomator.com) start your subscription (Free Trial available)  and add your partner code at plugin settings page.
1. You're done!
